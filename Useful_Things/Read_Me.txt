The Useful_Things directory is not necessary for ETLs and need not be kept in this relative location.  It is just other resources related to use of Pentaho Kettle that might be helpful.

Show_Update_Times_of_View.ktr - A Kettle transformation to be run in Spoon (the Kettle GUI).  It pulls update information about a dataset from Socrata APIs and presents it in a human-readable format.
